📎 Forensics / OSINT · Challenge #05

Phishing Payload

A PDF was sent via phishing email. It looks harmless, but something is embedded inside. Analyse the document and extract the hidden file.

Scenario

A user in your organisation received the email below and opened the attachment. The PDF rendered normally — but your SIEM flagged it as suspicious. Investigate the document, find what is hidden inside, and extract the flag.

Intercepted Phishing Email

Inbox — Mail Client
From:hr-noreply@acme-corp.support To:you@company.com Subject:⚠ ACTION REQUIRED: Q4 Benefits Update Date:Nov 22, 2024 — 03:14 UTC

Dear Employee,

Please review the attached HR document regarding your Q4 compensation update. Action is required before December 15.

The document contains embedded policy files that must be extracted and reviewed. Please open and review all attachments.

— HR Department, ACME Corp

Analysis Approach

1
Download document.pdf. Check metadata with exiftool — the Subject field contains a useful hint.
2
PDFs can contain embedded file attachments per the ISO 32000 spec. These are invisible in PDF viewers but detectable with forensic tools.
3
Use pdfdetach, binwalk, or Python pypdf to list and extract all embedded files from the PDF stream.
4
Read the extracted file — it contains the flag in plaintext near the bottom.

Extraction Methods

METHOD 1 — pdfdetach (poppler-utils)
# sudo apt install poppler-utils (Linux) $ pdfdetach -list document.pdf Embedded files: 1: secret_data.txt $ pdfdetach -saveall document.pdf Saving embedded file: secret_data.txt $ cat secret_data.txt ACME Corp — Classified Personnel File ... Internal Access Code: FlagVault{3mb3dd3d_f1l3_1ns1d3_pdf}
METHOD 2 — Python pypdf
# pip install pypdf $ python3 >>> from pypdf import PdfReader >>> r = PdfReader("document.pdf") >>> print(list(r.attachments.keys())) ['secret_data.txt'] >>> data = r.attachments['secret_data.txt'][0] >>> print(data.decode()) ACME Corp — Classified Personnel File ... FlagVault{3mb3dd3d_f1l3_1ns1d3_pdf}
METHOD 3 — binwalk (file carver)
# pip install binwalk OR sudo apt install binwalk $ binwalk document.pdf DECIMAL HEX DESCRIPTION 0 0x0 PDF document 2338 0x922 Zip archive (embedded) $ binwalk -e document.pdf && cat _document.pdf.extracted/*.txt FlagVault{3mb3dd3d_f1l3_1ns1d3_pdf}
METHOD 4 — strings (quickest)
$ strings document.pdf | grep FlagVault FlagVault{3mb3dd3d_f1l3_1ns1d3_pdf}
BONUS — PDF metadata (hint in Subject field)
$ exiftool document.pdf Title : Employee Benefits Q4 2024 Author : HR Department Subject : Check attachments - pdfdetach -saveall Keywords: embedded files attachment extract Creator : ACME HR Suite v3.1
